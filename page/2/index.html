报错：java.lang.NoClassDefFoundError: javax/xml/bind/JAXBException（运行环境：jdk10)

解决方法一：将JDK换成8或以下
解决办法二：导入下面4个jar包

  1. javax.activation-1.2.0.jar
  2. jaxb-api-2.3.0.jar
  3. jaxb-core-2.3.0.jar
  4. jaxb-impl-2.3.0.jar
    具体原因参考：https://stackoverflow.com/questions/43574426/how-to-resolve-java-lang-noclassdeffounderror-javax-xml-bind-jaxbexception-in-j

Java实现图片转字符画

发表于 2018-06-10 | 分类于 爬坑小项目 | 阅读次数:

一、实现思路：

  1. 1.获取彩色图片的RGB分量；
  2. 2.根据公式Gray = R*0.299 + G*0.587 + B*0.114,获取图像灰度值（范围在0~255）
  3. 3.不同灰度值用复杂度不同的字符输出即可

二、项目地址

https://github.com/fishrong/Pic2Char.git

三、项目相关说明

  1. 1.生成了window可执行文件，可脱离Java开发环境运行
  2. 2.命令行中直接输入桌面下的图片名称即可进行转换
  3. 3.转换后在桌面生成与图片同名的txt文件，并自动打开
  4. 4.可多次转换
  5. 5.设置最小字号才能看到效果
